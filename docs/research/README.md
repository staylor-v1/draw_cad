# Engineering Drawing Research Notes

This folder stores distilled research used to guide the Draw CAD development
dashboard and future drawing-recognition work. The intent is to preserve
pedigree for product and agent-design decisions so future sessions can build on
the same assumptions without repeating basic research.

## How To Use These Notes

Use these files as design constraints and acceptance criteria for segmentation,
OCR, and feedback UI work. They are not a replacement for paid standards such as
ASME Y14.1 or ISO 7200, but they capture the public-source reasoning behind the
current implementation.
